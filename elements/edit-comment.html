<edit-comment> ​

Usage ​

  1. If you haven't already, go through the setup for the module.

  2. Import the <edit-comment> element somewhere in the javascript side of your web-app like this:

js
import '@darksoil-studio/comments/dist/elements/edit-comment.js'
  1. Use it in the html side of your web-app like this:
html
<edit-comment .commentHash=${ commentHash }>
</edit-comment>
html
<edit-comment commentHash={ commentHash }>
</edit-comment>
html
<edit-comment [commentHash]="commentHash">
</edit-comment>
html
<edit-comment :commentHash="commentHash">
</edit-comment>
html
<edit-comment comment-hash={encodeHashToBase64(commentHash)}>
</edit-comment>

WARNING

Like all the elements in this module, <edit-comment> needs to be placed inside an initialized <comments-context>.
